# Richard Reukema

Cloud Architect | Event Driven Systems | AI Enabled Platforms

I design distributed systems where business events become operational signals. My work focuses on event driven architecture, AI assisted orchestration, and integration platforms that allow organizations to act on real operational signals rather than delayed reports.

## Platform Focus

Eventryx

The core event platform and runtime for publishing, routing, and processing operational events. Eventryx treats business events as first class system assets and enables organizations to build reactive systems that respond to real world activity.

SignalWeaver

The ecosystem and network layer built on top of Eventryx. SignalWeaver provides the public facing infrastructure for publishing, distributing, and subscribing to operational signals across organizations.
